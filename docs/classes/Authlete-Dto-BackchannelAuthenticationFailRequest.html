BackchannelAuthenticationFailRequest implements ArrayCopyable, Arrayable, Jsonable Uses ArrayTrait, JsonTrait

BackchannelAuthenticationFailRequest.php : 47

Request to Authlete's /api/backchannel/authentication/fail API.

The API prepares JSON that contains an error property and optionally others. The JSON should be used as the response body of the response which is returned to the client from the backchannel authentication endpoint.
